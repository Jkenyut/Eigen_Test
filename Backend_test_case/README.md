### install library

`$ npm install `

### aktifkan prisma

`npx prisma init` schema prisma
`npx prisma migrate dev` migrasi database
`npx prisma generate` generate prisma client

### menjalankan program

`npm run dev` menjalankan program
`npm run test` menjalankan test dengan 21 pass test
